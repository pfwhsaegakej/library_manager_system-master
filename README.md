# springboot-libraryManageSystem

>  图书管理系统 ，使用 SpringBoot 框架。

##  一、相关技术栈

1. **前端:** Thymeleaf、Layui、Ajax、JQuery
2. **后端:**  SpringBoot、MyBatis 
3. **开发环境:** jetBrains Intellij IDEA 、SpringBoot 2.3、Maven
4. **数据库:** MySQL、avicat
